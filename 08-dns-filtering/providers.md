# DNS Providers
> 🟢 Beginner

## Comparison
| Provider | Focus | Logs | Filtering |
|----------|-------|------|-----------|
| [Mullvad DNS](https://mullvad.net/dns) | Privacy | None | Optional |
| [Quad9](https://quad9.net) | Security | Minimal | Malware |
| [NextDNS](https://nextdns.io) | Customizable | Optional | Full |
| [ControlD](https://controld.com) | Flexible | Optional | Full |
| [AdGuard DNS](https://adguard-dns.io) | Ad blocking | Minimal | Ads/Trackers |

## Profiles
- **Maximum privacy:** Mullvad DNS - no logs, no customization needed
- **Security focus:** Quad9 - blocks malicious domains using threat intelligence
- **Power users:** NextDNS or ControlD - custom blocklists, per-device profiles
