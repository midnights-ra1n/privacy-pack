# Useful Resources

## Reference Sites
- [PrivacyGuides.org](https://www.privacyguides.org) - Comprehensive, community-maintained tool recommendations
- [EFF Surveillance Self-Defense](https://ssd.eff.org) - Practical guides from the EFF
- [The New Oil](https://thenewoil.org) - Beginner-friendly privacy guides
