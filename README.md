# privacy-pack

A pragmatic reference guide for protecting your privacy online - tools, software, self-hosting, DNS filtering, VPNs, and operational security.

> **Philosophy:** Not about paranoia. About informed decisions based on your threat model.

## Skill Levels
| Tag | Meaning |
|-----|---------|
| 🟢 Beginner | No technical knowledge required |
| 🟡 Intermediate | Some technical familiarity needed |
| 🔴 Advanced | Significant setup effort |
| ⚫ Expert | For high-risk individuals |
