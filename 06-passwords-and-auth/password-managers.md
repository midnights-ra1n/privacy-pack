# Password Managers
> 🟢 Beginner

## Why You Need One
Reusing passwords means one breach exposes all your accounts. A password manager lets you use unique, strong passwords everywhere.

## Recommended
| Manager | Type | Open Source | Notes |
|---------|------|-------------|-------|
| [Bitwarden](https://bitwarden.com) | Cloud | ✅ | Best for most users; generous free tier |
| [KeePassXC](https://keepassxc.org) | Local | ✅ | Offline vault, full control |
| [Vaultwarden](https://github.com/dani-garcia/vaultwarden) | Self-hosted | ✅ | Bitwarden-compatible server |

## Best Practices
- Use a long passphrase as master password (4–6 random words)
- Enable 2FA on your vault
- Back up your vault regularly
- Generate passwords at 20+ random characters per site
