# Contributing

## Guidelines
- No sponsored content, no affiliate links
- Cite sources (official docs, audits, reputable research)
- Flag outdated content with `> ⚠️ This section may be outdated`
- Be pragmatic real-world usability over theoretical perfection

## Tool Template
```markdown
## Tool Name
> 🟢/🟡/🔴/⚫

**What it is:** One sentence.
**Why it matters:** What threat does it address?

### Pros
- ...

### Cons
- ...

### Links
- [Official site](https://...)
- [Audit](https://...)
```

## Open Sections
- `05-operating-systems/android.md` - open to contributors with GrapheneOS/CalyxOS experience

## How to Submit
1. Fork the repo
2. Branch: `git checkout -b improve/section-name`
3. Follow the template
4. Open a pull request
