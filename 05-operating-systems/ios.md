# iOS Privacy
> 🟢 Beginner

## What iOS Does Well
- Strong app sandboxing
- App Tracking Transparency (apps must ask to track)
- On-device processing for Siri/Photos

## Limitations
- Closed-source
- Tied to Apple ID
- iCloud backups not E2EE by default

## Recommended Settings
- Settings → Privacy & Security → Tracking - disable "Allow Apps to Request to Track"
- Settings → Apple ID → iCloud → **Advanced Data Protection** - enable for E2EE backups
- Settings → Analytics & Improvements - disable all
- Review per-app permissions (location, contacts, microphone)
